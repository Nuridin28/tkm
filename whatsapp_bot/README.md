# WhatsApp Bot

Простой WhatsApp бот, который отвечает "привет" на любое сообщение.

## Установка

1. Установите зависимости:
```bash
cd whatsapp_bot
npm install
```

## Запуск

```bash
npm start
```

или

```bash
node bot.js
```

## Использование

1. При первом запуске бот сгенерирует QR-код в консоли
2. Откройте WhatsApp на вашем телефоне
3. Перейдите в Настройки → Связанные устройства → Связать устройство
4. Отсканируйте QR-код из консоли
5. После успешной авторизации бот будет готов к работе
6. Отправьте любое сообщение боту, и он ответит "привет"

## Примечания

- При первом запуске создается папка `auth` для хранения сессии
- Не удаляйте папку `auth`, иначе придется авторизовываться заново
- Бот работает через WhatsApp Web API
