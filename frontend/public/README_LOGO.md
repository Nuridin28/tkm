# Добавление логотипа Казахтелеком

Для добавления логотипа Казахтелеком:

1. Поместите файл логотипа в директорию `frontend/public/` с именем `logo.png`
2. Рекомендуемые размеры: 
   - Ширина: 200-300px
   - Высота: пропорционально ширине
   - Формат: PNG с прозрачным фоном (рекомендуется)

3. Логотип будет автоматически отображаться на всех страницах:
   - Страница входа (Login)
   - Админ панель
   - Панель департамента
   - Панель инженера
   - Панель оператора
   - Публичная поддержка

Если логотип не найден, будет отображаться текстовый вариант "КТ Казахтелеком".
